# StadiumPulse AI ⚽🏟️
**Challenge 4: Smart Stadiums & Tournament Operations**

StadiumPulse AI is a multi-agent generative AI orchestration network built for the FIFA World Cup 2026 to optimize stadium operations, provide real-time decision support, and elevate the global fan experience.

## 🤖 Gen AI Architecture & Features
- **Groq SDK & LLM Orchestration:** Powering our real-time operational dashboard pipeline (`/routes/ask.js`) to handle crowd density logs, transit alerts, and accessibility support.
- **Local Text Embeddings (`@xenova/transformers`):** Running lightweight token features locally within the backend layer (`/routes/semanticSearch.js`) to support semantic vector parsing.

## 🛠️ Tech Stack
- **Backend Environment:** Node.js, Express
- **AI Pipelines:** Groq API Cloud Client, Xenova Transformers Pipeline
- **Dependencies Layer:** CORS, Dotenv
